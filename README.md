# Scheduler

Приложение для планирования задач.

## Возможности

- создание задач;
- просмотр списка задач;
- редактирование задач;
- удаление задач;
- отметка задачи выполненной;
- поддержка повторяющихся задач.

## Задания со звёздочкой

Реализован поиск задач:

- по заголовку и комментарию;
- по дате.

## Запуск проекта

Установите переменные окружения:

```bash
export TODO_PORT=7540
export TODO_DBFILE=scheduler.db
```

Запустите приложение:

```bash
go run .
```

После запуска откройте в браузере:

```
http://localhost:7540
```

## Запуск тестов

Основные тесты:

```bash
go test ./tests
```
